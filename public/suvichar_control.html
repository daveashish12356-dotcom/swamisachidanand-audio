ઉપર ON/OFF દબાવો, નીચે "સેવ કરો" દબાવો — GitHub નહીં ખુલે.

સુવિચાર અહીં કોપી-પેસ્ટ કરો (અથવા એડિટ કરો)

નીચેનો JSON બદલો. suvicharEnabled: true = ચાલુ, false = બંધ. પછી "સેવ કરો" દબાવો.

Token (એક વાર): GitHub → Settings → Developer settings → Personal access tokens → Generate new token (classic), repo ચેક કરો. Token ઉપર પેસ્ટ કરો, પછી "સેવ કરો" દબાવો.

એપ: સેવ પછી એપમાં વિડિઓ ટેબ ખોલો/રિફ્રેશ કરો.
