ભાર્ગવ સુવિચાર – કંટ્રોલ

એપમાં સુવિચાર દેખાશે કે નહીં તે અહીંથી કંટ્રોલ કરો.

હાલની સ્થિતિ (એપમાં શું દેખાશે)
ચેક થાય છે...
સુવિચાર ON કરો સુવિચાર OFF કરો
એક ક્લિકથી શું થાય:
• ON કરો દબાવો → GitHub પર ફાઇલ ખુલશે → "suvicharEnabled": false ને true કરો → નીચે "Commit changes" દબાવો. એપમાં સુવિચાર દેખાશે.
• OFF કરો દબાવો → ફાઇલમાં "suvicharEnabled": true ને false કરો → Commit કરો. એપમાં સુવિચાર છુપાઈ જશે.

એપ સાથે: આ પેજ પર જે બદલાવ કરો (Commit પછી) એપમાં ઓટોમેટિક લાગુ થશે. વિડિઓ ટેબ ખોલો અથવા નીચે ખેંચી રિફ્રેશ કરો.
