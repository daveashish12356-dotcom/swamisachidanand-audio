સુવિચાર અહીં કોપી-પેસ્ટ કરો

નીચેનો JSON એડિટ કરો અથવા નવો પેસ્ટ કરો. suvicharEnabled: true = ચાલુ, false = બંધ. "સેવ કરો" દબાવો — GitHub પર જવાની જરૂર નથી.

Token (એક વાર): GitHub → તમારા પ્રોફાઇલ → Settings → Developer settings → Personal access tokens → Generate new token (classic). repo ચેક કરો, token કોપી કરીને ઉપર પેસ્ટ કરો. પછી "સેવ કરો" દબાવો — GitHub પેજ ખોલવું પડશે નહીં.

એપ સાથે: "સેવ કરો" પછી એપમાં ઓટોમેટિક લાગુ થશે. વિડિઓ ટેબ ખોલો અથવા નીચે ખેંચી રિફ્રેશ કરો.
