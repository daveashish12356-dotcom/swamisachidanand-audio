૧. અહીં પેસ્ટ કરો — જે લખ્યું એમ જ (ટાઇટલ, વાક્ય, 📚 સ્રોત)

શ્રી જે લખે છે એ સંપૂર્ણ કોપી કરી અહીં પેસ્ટ કરો. 📚 વાળી લાઇન ઓટોમેટિક સ્રોત બનશે. પછી «ઉમેરો» દબાવો.

૨. સેવ કરો

ટિપ: ફક્ત સુવિચાર ટેક્સ્ટ અહીં પેસ્ટ કરી «સેવ કરો» દબાવો — ઓટો લગાવશે.
પુરાનો સુવિચાર દેખાય તો Ctrl+Shift+R દબાવી પેજ ફરી લોડ કરો.

GitHub પર token બનાવો (એક ક્લિક)
Token લગાવાઈ ગયું. ફરીથી કંઈ લગાવવું પડશે નહીં — ફક્ત ON/OFF બદલો અને «સેવ કરો» દબાવો.
પેસ્ટ ફક્ત એટલું: કોઈ પણ જગ્યાથી સુવિચાર કોપી કરી અહીં (ઉપરના પેસ્ટ બોક્સ અથવા નીચેના બોક્સ) પેસ્ટ કરો → «સેવ કરો» દબાવો. બાકી ઓટો.
જો "અમાન્ય JSON" દેખાય: પેજ પુરાનો લોડ થયો છે. Ctrl+Shift+R (કે Mac પર Cmd+Shift+R) દબાવી પેજ ફરી લોડ કરો.

સુરક્ષા: Token ક્યારેય વેબસાઇટ/કોડ માં લગાવશો નહીં — ફક્ત અહીં બોક્સ માં પેસ્ટ કરો.
એપ: સેવ પછી એપમાં વિડિઓ ટેબ ખોલો/રિફ્રેશ કરો.
