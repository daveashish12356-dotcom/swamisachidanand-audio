ભાર્ગવ સુવિચાર – ON / OFF

એપમાં સુવિચાર દેખાશે કે નહીં તે અહીંથી કંટ્રોલ કરો.

સુવિચાર ON સુવિચાર OFF
કેવી રીતે ચાલુ/બંધ કરવું:
• ON – ઉપર બટન દબાવો, GitHub પર ફાઇલ ખુલશે. "suvicharEnabled": false ને "suvicharEnabled": true કરો, પછી Commit કરો.
• OFF – "suvicharEnabled": true ને "suvicharEnabled": false કરો, પછી Commit કરો.
